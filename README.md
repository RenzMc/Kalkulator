⚡ Quick Start  
  
🔧 Instalasi

git clone https://github.com/RenzMc/Kalkulator.git  
  
# Masuk ke direktori  
cd Kalkulator  
  
# Buka di browser  
open index.html

# Menggunakan Python  
python -m http.server 8000  
  
# Menggunakan Node.js  
npx live-server  
  
# Menggunakan PHP  
php -S localhost:8000

1. Fork repository ini

5. Save dan tunggu deployment

**Basic Operations**

inputFunction('log') → Result: 1  
  
  
**Physics Formulas**  
  
// Menggunakan rumus fisika  
insertPhysicsFormula('F = m*a')  
// Input nilai dan hitung  
  
  
⌨️ Keyboard Shortcuts  
  
Angka 0-9     → Input angka  
+ - * /       → Operator matematika  
Enter atau =  → Hitung hasil

( )           → Kurung (mode scientific)

🐛 Troubleshooting  
  
Common Issues  
  
**Voice Recognition tidak bekerja**  
  
✓ Pastikan browser mendukung Web Speech API  
✓ Berikan permission microphone  
✓ Gunakan HTTPS atau localhost  
✓ Check browser compatibility  
  
  
**Animasi lag atau tidak smooth**  
  
✓ Reduce particle count di createBackgroundAnimation()  
✓ Disable hardware acceleration jika perlu  
✓ Check GPU performance  
✓ Update browser ke versi terbaru  
  
  
**Layout broken di mobile**  
  
✓ Check viewport meta tag  
✓ Test di different screen sizes  
✓ Verify media queries  
✓ Check touch event handlers
